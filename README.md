# Eco Vibe

Eco Vibe is a sustainable storefront demo built with Astro, React, and Tailwind. The project focuses on polished product browsing, cart interactions, and a clean nature-inspired aesthetic.

## Highlights

- Product catalog and slide-out cart flow
- Responsive storefront UI with reusable components
- Astro build with light React interactivity where needed

## Stack

- Astro
- React
- Tailwind CSS

## Quick start

```bash
bun install
bun run dev
```

Build and check:

```bash
bun run check
bun run build
```
